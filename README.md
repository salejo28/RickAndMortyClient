# Rick And Morty Client

## Steps

### Download the code

```
git clone https://github.com/santi280403/RickAndMortyClient.git
```

### Navigate

```
cd RickAndMortyClient
```

### Install dependencies

```
npm install
```

o

```
yarn
```

### Run the project (dev)

```
npm run dev
```

### Run the project (prod)

```
npm run build
```

y

```
npm start
```
